# Michael

基于 Harness Engineering 哲学构建的 ICT 交易分析系统。

## 项目背景

Michael 是两个前身项目的继承者：

| 项目 | 架构 | 定位 |
|------|------|------|
| [ICT_Advisor](https://github.com/patrick369-ai/ICT_Advisor) | Bash + Claude CLI "提示软件" | 原型验证，生产运行 |
| [ict-advisor-refactory](https://github.com/patrick369-ai/ict-advisor-refactory) | Python 5 层流水线 | 工程化重构 |

两个项目各有优劣，Michael 的目标是基于 Harness Engineering 十原则，从中取长补短，构建一个既有工程纪律又保持业务深度的系统。
